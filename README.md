# LLM Key Manager

**Zentrale, sichere Verwaltung deiner LLM API Keys**

## Features
- Registrieren / Login
- Keys speichern (AES-verschlüsselt)
- Ein-Klick-Test (OpenAI, Grok)
- Löschen
- Schöne UI mit Tailwind

## Live Demo
[http://localhost:5000](http://localhost:5000) (lokal)

```bash
npm install
npm run dev
